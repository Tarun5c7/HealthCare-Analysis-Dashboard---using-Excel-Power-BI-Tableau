**🏥 Healthcare Analysis Dashboard **
** 📊 Project Overview**

The **Healthcare Analytics Dashboard** is a comprehensive Business Intelligence project developed using:
 * Microsoft Excel (with Power Pivot)
 * Microsoft Power BI
 * Tableau Desktop

This project analyzes healthcare operational and patient data to monitor performance, patient trends, visit patterns, and departmental efficiency.
The dashboards transform raw healthcare data into actionable insights, enabling data-driven healthcare management decisions.

This project demonstrates practical expertise in data cleaning, data modeling, DAX calculations, calculated fields, and interactive dashboard design across multiple BI platforms.

**🎯 Objectives**

* ✔ Monitor overall healthcare performance
* ✔ Analyze patient and visit trends
* ✔ Track operational efficiency
* ✔ Enable interactive data exploration
* ✔ Support data-driven decision-making

**🛠 Tools & Technologies Used**

1.📗 Excel (Power Pivot)
      * Pivot Tables & Pivot Charts
      * Power Pivot Data Model
      * DAX Measures
      * Slicers
      * Structured Data Tables 
2. 📊 Power BI
      * Power Query (Data Transformation)
      * Data Modeling & Relationships
      * DAX (Data Analysis Expressions)
      * KPI Cards & Interactive Visuals
3. 📈 Tableau
      * Calculated Fields
      * Parameters
      * Dashboard Actions
      * Interactive Filters
      * Visual Analytics

📁 **Dataset Description**

The dataset represents healthcare operational data and includes:

*🔸 Patient details
*🔸 Visit records
*🔸 Appointment information
*🔸 Follow-up status
*🔸 Demographic data

⚠ Dataset is simulated/anonymized for academic and portfolio use.

**🔄 Data Preparation**

✔ Removed duplicates and handled missing values
✔ Converted raw data into structured tables
✔ Built relationships using Data Model
✔ Created calculated measures using DAX

**📐 Dashboard Features**

📌 KPI cards for performance overview
📌 Pivot charts for trend analysis
📌 Slicers for dynamic filtering
📌 Interactive visuals for drill-down insights
📌 Clean and structured dashboard layout

**🔍 Insights Generated**

📊 Patient and visit distribution patterns
📊 Trend analysis over time
📊 Operational performance overview
📊 Interactive filtering for deeper analysis

**🚀 How to Use the Dashboard **
1.Download the respective dashboard file:
  *.xlsx for Excel
  *.pbix for Power BI
  *.twbx for Tableau
2.Open using:
  *Microsoft Excel (Power Pivot enabled)
  *Power BI Desktop
  *Tableau Desktop
3.Use slicers and filters to explore insights interactively.
